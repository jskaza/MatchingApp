This is a proof-of-concept app for N-to-1 propensity score matching. Begin by uploading a tab-delimted .txtfile with the following characteristics:

  • Row 1 contains column names
  • One column denotes case/control status as a string (e.g., "Case" or "Control")
  • One column serves as an identifier (e.g., a unique ID). The same ID should not serve as both a Case and a Control.
